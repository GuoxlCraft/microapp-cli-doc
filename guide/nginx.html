# 部署配置

这里以nginx的部署配置为例。

首先主应用和子应用打包出来这之后可以根据应用名称参考以下目录结构。

.
├── mian # 主应用
├── npmScript # 快捷脚本配置
├── apps # 子应用目录
    ├── App-MicoAppOne # 范例子应用1
    └── App-MicoAppTwo # 范例子应用2

# nginx配置

配置参考下列配置。需要注意的是应用需要配置 try_files 在应用匹配不到路径的时候指向项目的index.html。


   server {
        listen 8088;
        server_name localhost;
        #client_max_body_size 16m;
        #charset koi8-r;

        location / {
            root html;
            index main/index.html index.htm;
            try_files $uri $uri/ /main/index.html;
        }

        location /App-MicoAppOne{
            alias html/App-MicoAppOne/;
            index index.html index.htm;
            try_files $uri $uri/ /App-MicoAppOne/index.html;
        }

	    location /App-MicoAppTwo{
            alias html/App-MicoAppTwo/;
            index index.html index.htm;
            try_files $uri $uri/ /App-MicoAppTwo/index.html;
        }
