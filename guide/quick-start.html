# 快速上手

提示

本项目需要Node.js (opens new window)版本>=14

# 代码获取

注意

注意存放代码的目录及所有父级目录不能存在中文、韩文、日文以及空格，否则安装依赖后启动会出错。

# clone 代码
git clone https://github.com/GuoxlCraft/microapp-cli.git

# 安装 Node.js

如果您电脑未安装Node.js (opens new window)，请安装它。

# 验证

# 出现相应npm版本即可
npm -v

# 出现相应node版本即可
node -v

# 快速启动

分别进入根目录下的main目录和apps下的各个子应用所在的目录安装依赖

推荐使用cnpm (opens new window)进行依赖安装。

在项目对应的目录下，打开命令窗口执行，耐心等待安装完成即可

# 安装依赖

# 安装依赖
cnpm i

另外，您也可以在根目录下使用脚本命令为主应用和所有子应用安装依赖。

# 主应用和当前子应用安装依赖
npm run all:install

注意

目前脚本命令只是服务于现有的目录，在开发过程中应当根据具体的项目结构修改脚本命令。具体的配置方法将在进阶栏目说明。

# 运行项目

# 运行项目
npm run serve

另外，您也可以在根目录下使用脚本命令快速开启主应用和所有子应用的本地服务。

# 主应用和当前子应用安装依赖
npm run all:start

注意

目前脚本命令只是服务于现有的目录，在开发过程中应当根据具体的项目结构修改脚本命令。具体的配置方法将在进阶栏目说明。
